ScrStart.exe

This Win32 executable can be ran to start a Windows screen saver. When used in conjunction
with a passworded screen saver, this program emulates the Lock Computer feature of Windows NT.

Note: You must select a screen saver from the Display Properties of your computer or this
program will fail to function as expected.

The binary contained within is distributed AS IS and no support is given for it. If you don't like it,
change the source and compile it yourself.
